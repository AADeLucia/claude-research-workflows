# claude-research-workflows
Collection of Claude Code commands for a hands-on AI-assisted research workflow.

## Getting Started
To use the commands in this repository, you can either symlink the `.claude/commands` directory or add it as a git submodule to your existing Claude setup.

```bash
# Option 1: symlink
ln -s ~/path/to/claude-skills/.claude/commands .claude/commands

# Option 2: submodule
git submodule add https://github.com/AADeLucia/claude-research-workflows .claude
```
